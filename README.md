# Ocean View Resort - Reservation System

This is a Java Web Application (JSP/Servlet) built for the Ocean View Resort management system.

### Features
- **User Login:** Secure login for Admins and Staff.
- **Dashboard:** Real-time statistics (Revenue, Pending Bookings).
- **Reservations:** Add, View, Edit, Search, and Delete bookings.
- **Reports:** Visual bar chart for monthly bookings.
- **Tech Stack:** Java, JSP, MySQL, Apache Tomcat 10, Bootstrap 5.

### How to Run
1. Import the `ocean_view_resort.sql` into MySQL.
2. Update `DBConnection.java` with your database credentials.
3. Run on Apache Tomcat 10.
